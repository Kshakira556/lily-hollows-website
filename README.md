# Lily Hollows — Official Website

## Project Info

This is the official website for Lily Hollows — singer-songwriter, performer, and actress.

## How can I edit this code?

You can work on this project locally using your preferred IDE.

### Requirements

- Node.js
- npm (or yarn/pnpm)

### Getting Started

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Step 3: Install dependencies
npm install

# Step 4: Start the development server
npm run dev
````

## Tech Stack

This project is built with:

* Vite
* TypeScript
* React
* shadcn-ui
* Tailwind CSS

## Deployment

Build the project for production:

```sh
npm run build
```

Preview the production build locally:

```sh
npm run preview
```

You can deploy the `dist` folder to any static hosting provider (e.g. Vercel, Netlify, or a custom server).
